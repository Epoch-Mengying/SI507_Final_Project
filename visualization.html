Welcome to Mengying Zhang's Yelp Resraurants Interactive Map!

This interactive map is generated using Tableau. The original data is retrieved from Yelp Search API and saved in csv file. This is part of my SI507 final project. For the whole project specification, please visit here. The map shows:

  • restaurant name
  • categories
  • location
  • Ratings, and
  • Review counts
Enjoy!
